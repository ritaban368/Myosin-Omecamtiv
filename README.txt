The input files, structure files and scripts used for the wild type myosin vi, mutated myosin vi and omecamtiv bound myosin vi are included here.

CHARMM force field parameters are included.

The link of NCBI protein database and the source of MUSCLE (for multiple sequence alignment)included.

Helix properties calculation have been performed using HELIQUEST (https://heliquest.ipmc.cnrs.fr/index.html).
